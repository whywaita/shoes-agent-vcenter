# vmware

## Generate

`openapi` stored models for REST API. This directory generated by OpenAPI spec using `vmware/vmware-openapi-generator` and `openapi-generator-cli`.

```
$ git clone https://github.com/vmware/vmware-openapi-generator
$ python vmsgen.py -vc '<vCenter IP>' -o ./dist -k

$ docker run --rm -v ${PWD}:/local openapitools/openapi-generator-cli generate -i /local/dist/vcenter.json -g go --skip-validate-spec --global-property models --additional-properties=structPrefix=true,enumClassPrefix=true -o /local/output/
$ cp -a ./output ${path_to_this}/openapi
```

REST API of vCenter has a query parameter in endpoint (e.g. `https://{api_host}/rest/vcenter/vm?action=clone`). But this format is invalid in OpenAPI.
So my repository is only using model definition.
